BillPouch

A P2P social distributed filesystem — written in Rust.

GitHub API Docs

The Pelican Metaphor

BillPouch takes its name from the pelican — a bird that hunts, stores, and shares. Every user joins the network by running one or more services:

🦤

Pouch storage

Bids a portion of your local disk into the network. You say "I have 10 GB" and the network can store data on your machine.

📂

Bill file I/O

Your personal window into the distributed filesystem. Read and write files that are spread across Pouch nodes in the network.

📡

Post relay

A pure routing node — contributes only CPU and bandwidth. Bridges peers that can't directly connect (NAT, firewalls).

How nodes communicate

mDNS

Zero-config local network discovery. Like AirDrop — finds peers on LAN automatically.

Kademlia DHT

Distributed hash table for peer discovery across the internet.

Gossipsub

Every node periodically broadcasts its NodeInfo. All nodes maintain a local view of the network.

Quick Start

Build from source

git clone https://github.com/Onheiron/BillPouch.git
cd BillPouch
cargo build --release

Create your identity

bp login --alias "your-name"

Start a storage node

bp hatch pouch --network my-network --storage-bytes 10737418240

Inspect the flock

bp flock
